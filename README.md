<h1 align="center">JSON flattener util</h1>
<p>Just a quick util I threw together to flatten json into slightly more friendly txt files.<br>
  It's mostly intended for sharing around data to less tech-savvy people that might not know how to deal with a .json file without having to manually rewrite the entire thing.</p>
<p>This util is provided as is first and foremost, that being said however, if you find any bugs you're welcome to report them.<br>
  But since this is my first project, and first proper use of github I make no promises.</p>
